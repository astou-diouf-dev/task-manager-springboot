# Task Manager

Application web de gestion des tâches développée avec Spring Boot et Thymeleaf.

## Fonctionnalités
- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Marquer une tâche comme terminée
- Afficher la liste des tâches

## Technologies utilisées
- Java
- Spring Boot
- Thymeleaf
- MySQL
- Bootstrap

## Lancer l'application

1. Cloner le projet
2. Ouvrir dans un IDE (IntelliJ ou VS Code)
3. Lancer l'application
4. Ouvrir dans le navigateur :

http://localhost:8080
